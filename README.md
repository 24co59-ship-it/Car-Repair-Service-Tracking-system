# Car-Repair-Service-Tracking-system
# GearHaus — PHP + HTML + SQL Version

This is the converted version of the GearHaus / Car Repair Service Tracking System using:

- HTML
- CSS
- PHP backend
- MySQL database
- No React
- No FastAPI
- No Node.js required

## How to run in XAMPP

1. Copy the folder `gearhaus_php_mysql` into:

   `C:\xampp\htdocs\gearhaus_php_mysql`

2. Start XAMPP:

   - Apache
   - MySQL

3. Open phpMyAdmin:

   `http://localhost/phpmyadmin`

4. Import:

   `database/schema.sql`

5. Open:

   `http://localhost/gearhaus_php_mysql`
